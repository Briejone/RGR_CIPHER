# RGR_CIPHER — Реализация криптографических алгоритмов

Репозиторий с реализацией классических и современных шифров в рамках расчётно-графической работы (РГР).  
Содержит как симметричные, так и асимметричные алгоритмы шифрования, а также вспомогательные утилиты для работы с ключами и файлами.

## 📁 Содержание

| Название | Тип | Описание |
|:--------:|:---:|:----------|
| **caesar** | Симметричный | Шифр Цезаря со сдвигом (поддержка русского/английского алфавита) |
| **vigenere** | Симметричный | Шифр Виженера с возможностью автоключа |
| **xor** | Симметричный | Побайтовое XOR-шифрование с заданным ключом |
| **rsa** | Асимметричный | Алгоритм RSA с генерацией простых чисел и ключей |
| **aes** | Симметричный | Реализация AES-128/256 в режиме ECB/CBC |
| **des** | Симметричный | Алгоритм DES (учебная реализация) |
| **hash** | Хеш-функции | SHA-256, MD5 (для демонстрации) |
| **utils** | Вспомогательные | Генерация ключей, работа с файлами, base64 |

*Актуальный список уточняйте в исходных файлах репозитория.*

## 🛠️ Требования

- Компилятор с поддержкой **C++17** (g++, clang++, MSVC)
- **CMake** 3.10+
- (Опционально) **OpenSSL** для сравнения с эталонными реализациями
